O que aprendi nos meus primeiros 6 meses estudando programação

Publicado em 1 de Julho de 2025

Quando comecei a estudar programação, tudo parecia muito confuso. Termos como “flexbox”, “margin”, “git” e até “display: block” me assustavam. Mas com o tempo e muita prática, fui ganhando confiança.

Descobri que consistência é mais importante que velocidade. Estudar todos os dias, mesmo que só um pouco, me ajudou a manter o foco. E encontrar formas visuais de entender o CSS mudou meu jeito de aprender.

Hoje, consigo montar sites simples sozinha, entendo HTML, CSS e JavaScript, e estou criando meu próprio portfólio. Ainda há muito o que aprender, mas agora sei que sou capaz!

Como criei meu primeiro portfólio do zero

Publicado em 10 de Julho de 2025

Criar um portfólio foi um desafio incrível e uma ótima forma de consolidar tudo o que aprendi. Usei HTML e CSS para estruturar e estilizar, e um pouco de JavaScript para interatividade básica.

Como fazer um botão com efeito hover no CSS

Publicado em 20 de Julho de 2025

Criar botões interativos é um jeito fácil de melhorar a usabilidade do seu site. Hoje vou mostrar um exemplo simples de botão com efeito hover usando CSS puro. 


.button {
  background-color: #505EFF;
  color: white;
  padding: 12px 25px;
  border-radius: 8px;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #404DDB;
}

Você pode usar essa ideia para deixar seu site mais bonito e com uma experiência melhor para o usuário.
